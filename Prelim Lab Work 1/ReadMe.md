# Bubble Sort Empirical Analysis

This project implements the Bubble Sort algorithm in Python to empirically analyze its performance on a dataset of integers.

## Description

The script automatically generates a dataset of random integers (if `dataset.txt` is missing), sorts them using an optimized Bubble Sort algorithm (ascending), and outputs the execution time and the sorted array.

## Requirements

*   Python 3.x
*   No external libraries required.

## How to Run

1.  Ensure `bubble_sort.py` is in your directory.
2.  Open a terminal or command prompt in that folder.
3.  Run the script using:
    ```bash
    python bubble_sort.py
    ```

## Output

The program will:
1.  Generate `dataset.txt` (if needed).
2.  Display status messages.
3.  Output the full sorted array.
4.  Report execution time in seconds.
